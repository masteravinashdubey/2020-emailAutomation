How to remake and use

  1. For changing the database settings go to settings.py file in django_backend/app/app folder there you will find
          DATABASES = {
           'default': {
            'ENGINE': 'django.db.backends.postgresql',
            'NAME': 'elsi',
            'USER':'postgres',
            'PASSWORD':'1234',
            'HOST':'localhost'
           }
         }
    Here ENGINE depicts the database used ,NAME is the database name, USER and PASSWORD are postgres pgAdmin username and password, HOST denotes server.

  2. For setting up the email used for sending mails, get the client id and client secret from developers console google and replace them with the client id and secret present in the credentials.json file (present in django_backend/app/assets folder) and also delete the pickle.token file present in same directory . Run the program and allow sending an email it will ask for permissions grant them and a new pickle.token file is generated and the mails will then be sent with the new email provided (For in-depth reference visit this link ) .
  3. To truncate the tables and import your own data, type one among these commands in psql shell.
    • If user wants a simple truncate to happen
      TRUNCATE table_name1,table_name2,....;
    • If user wants not only to truncate but also reset associated sequence generator(auto-incrementing field of the table)
      TRUNCATE TABLE table_name RESTART IDENTITY;
