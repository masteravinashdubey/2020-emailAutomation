views.CreateDraft(service, user_id, message_body)¶

Create and insert a draft email. Print the returned draft’s message and id.

Parameters

  • service – Authorized Gmail API service instance.

  • user_id – User’s email address. The special value “me” can be used to indicate the authenticated user.

  • message_body – The body of the email message, including headers.

Returns

Draft object, including draft id and message meta data.

views.ListMessagesWithLabels(service, user_id, label_ids=[])¶

List all Messages of the user’s mailbox with label_ids applied.

Parameters

  • service – Authorized Gmail API service instance.

  • user_id – User’s email address. The special value “me” can be used to indicate the authenticated user.

  • label_ids – Only return Messages with these labelIds applied.

Returns

List of Messages that have all required Labels applied. Note that the returned list contains Message IDs, you must use get with the appropriate id to get the details of a Message.
